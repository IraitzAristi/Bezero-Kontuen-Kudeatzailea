# Bezero Kontuen Kudeatzailea

C#-en garatutako programa, AllSecurity enpresako bezeroen kontuak eta pasahitzak kudeatzeko diseinatuta dago.

## Ezaugarriak
- Pasahitzen Kudeaketa
- Bilaketa Azkarra
- Kontu Motak (Gmail, Facebook, Instagram, Amazon...)
- Kontsolako interfazea

## Teknologiak
- C# .NET
- Array-ak datuak gordetzeko eta kontuen biltegiratzerako
- Datu Egiturak
- Datu motak (Bool, string, int...)
- Moduloak
- Kondizionalak (IF-ELSE IF-ELSE, SWITCH-CASE)

## Egoera
Proiektu hau garapenean dago.
